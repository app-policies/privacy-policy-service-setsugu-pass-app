# サービス接遇Pass プライバシーポリシー

制定日: 2026年6月18日
最終更新日: 2026年6月18日

公開URL: https://github.com/app-policies/privacy-policy-service-setsugu-pass-app

## 1. 基本方針

サービス接遇Pass（以下「本アプリ」）は、サービス接遇検定2級・3級の学習を支援するアプリです。本アプリでは、学習機能、広告表示、アプリ内課金、品質改善に必要な範囲で情報を取り扱います。

本アプリは、利用者の氏名、住所、電話番号、メールアドレス、勤務先、学校名、顔写真、音声、連絡先、位置情報を、アプリ独自の会員登録情報として収集しません。

## 2. 本アプリ内に保存される情報

本アプリは、学習機能を提供するため、以下の情報を端末内に保存します。

- 選択した対象級（3級または2級）
- 解答履歴
- 正誤情報
- 苦手問題の状態
- 学習日数、連続学習日数
- 文字サイズなどの設定
- プレミアム機能の有効状態

これらの学習データは、原則として利用者の端末内に保存されます。利用者は、アプリ内の設定画面から学習データをリセットできます。

## 3. 広告配信について

本アプリは、無料利用時に広告を表示するため、Google Mobile Ads SDKを利用します。広告配信の過程で、Googleまたは広告配信事業者が、広告識別子、端末情報、広告表示や広告への反応に関する情報を取得する場合があります。

取得される情報、利用目的、管理方法については、Googleのポリシーをご確認ください。

- Google プライバシーポリシー: https://policies.google.com/privacy
- Googleによる広告でのデータ利用: https://policies.google.com/technologies/ads

プレミアム機能を購入した場合、アプリ内広告は非表示になります。

## 4. アプリ内課金について

本アプリは、プレミアム機能の購入状態を確認するため、Appleのアプリ内課金およびRevenueCatを利用します。

課金処理はAppleが提供する仕組みにより行われます。本アプリの開発者が、利用者のクレジットカード番号、銀行口座情報、Apple IDのパスワードを取得することはありません。

RevenueCatでは、購入状態の確認、購入情報の復元、不正利用防止、課金機能の維持に必要な範囲で、購入に関する識別子やアプリ利用環境に関する情報が取り扱われる場合があります。

- RevenueCat Privacy Policy: https://www.revenuecat.com/privacy
- Apple Privacy Policy: https://www.apple.com/legal/privacy/

## 5. 診断情報・クラッシュ情報

本アプリの品質維持と不具合調査のため、Appleが提供する診断情報、クラッシュログ、パフォーマンス情報を参照する場合があります。これらの情報は、個別の利用者を直接特定する目的では利用しません。

## 6. お問い合わせ時の情報

利用者がメールで問い合わせを行う場合、返信および調査に必要な範囲で、メールアドレス、問い合わせ本文、利用者が任意で記載した情報を取り扱います。

問い合わせで取得した情報は、問い合わせ対応、本人への連絡、問題調査、品質改善のために利用し、不要になった時点で適切に削除します。

## 7. 第三者提供

本アプリは、法令に基づく場合を除き、利用者を直接特定できる個人情報を第三者へ販売または提供しません。

ただし、広告配信、アプリ内課金、診断情報の取得のために、Google、Apple、RevenueCatなどの外部サービスが、それぞれのポリシーに基づいて情報を取り扱う場合があります。

## 8. 利用者による管理

利用者は、端末の設定またはアプリ内の設定画面から、次の操作を行えます。

- 広告識別子の制限またはリセット
- 学習データのリセット
- 購入情報の復元
- アプリの削除による端末内データの削除

端末、OS、App Store、GoogleまたはAppleの設定に関する詳細は、各社が提供する案内をご確認ください。

## 9. 未成年の利用

未成年の方が本アプリを利用する場合は、保護者の同意を得たうえで利用してください。アプリ内課金を行う場合も、保護者またはApple ID管理者の承認を得てください。

## 10. 免責および非公式アプリであること

本アプリは、公益財団法人 実務技能検定協会、サービス接遇検定、その他の試験実施団体が提供、認定、監修、推薦する公式アプリではありません。

本アプリの問題、解説、学習機能は、公開されている出題範囲や問題例の傾向を踏まえて独自に作成したものです。公式問題、過去問題、公式問題集、公式テキストの本文を収録するものではありません。

## 11. プライバシーポリシーの変更

本ポリシーは、法令、外部サービスの仕様変更、アプリ機能の変更に応じて改定する場合があります。重要な変更がある場合は、アプリ内または公開URL上で確認できる形で告知します。
