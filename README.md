# サービス接遇Pass プライバシーポリシー

制定日: 2026年6月18日

サービス接遇Pass（以下「本アプリ」）は、学習機能、広告表示、アプリ内課金を提供するために、以下の外部サービスを利用する場合があります。

## 利用する外部サービス

- Google Mobile Ads
- RevenueCat
- Apple In-App Purchase
- Apple標準の診断・クラッシュ情報

## 取得される可能性のある情報

- 広告識別子
- 購入状態
- アプリ利用状況に関する情報
- クラッシュログ、診断情報

本アプリは、氏名、住所、電話番号、メール本文などを独自に収集することを目的としていません。
